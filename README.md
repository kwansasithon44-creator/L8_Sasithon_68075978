วิชา ICT12367  จัดทำโดย นางสาวศศิธร เสาร์ประโคน
   โปรเจกต์นี้เป็นการพัฒนาเว็บไซต์ร้านกาแฟด้วย Django โดยใช้แนวคิด Template Inheritance ผ่านไฟล์ base.html เพื่อกำหนดโครงสร้างหลักของเว็บไซต์ เช่น Navbar, Footer และการเชื่อมต่อ Bootstrap จากนั้นสร้างหน้า index.html, form.html และ about.html ให้สืบทอดโครงสร้างจากไฟล์หลักดังกล่าว
มีการออกแบบหน้าเว็บด้วย Bootstrap 5 ให้มีความสวยงามและเป็นธีมร้านกาแฟโทนอบอุ่น พร้อมปรับแต่ง CSS เพิ่มเติม เช่น Hero Section (ภาพพื้นหลังพร้อมข้อความกลางภาพ), Card Layout, ฟอร์มจองโต๊ะ, Gallery รูปภาพ และการจัดวางแบบ Responsive เพื่อให้รองรับทุกขนาดหน้าจอ
เป็นการฝึกการจัดโครงสร้างเว็บ การออกแบบ UI และการใช้งาน Django Template อย่างเป็นระบบ
